# pyncmd
[PyNCM](https://github.com/greats3an/pyncm) 的Web前端，旨在共享 CVIP 账户的音乐特权

## 注意
下载所调用的eapi实为网易云web端播放用api，故下载不会被计数

## 使用
    pyncmd.py [手机号] [密码] --message [附加信息] --port [端口号]
